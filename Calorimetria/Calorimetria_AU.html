Teoria - Calorimetria

Calor Sensível

Q = m × c × ΔT

Onde:

  • Q = quantidade de calor (cal ou J)
  • m = massa (g)
  • c = calor específico (cal/g°C)
  • ΔT = variação de temperatura (°C)

Calor Latente

Q = m × L

Onde:

  • Q = quantidade de calor (cal ou J)
  • m = massa (g)
  • L = calor latente (cal/g)

Calor latente de fusão do gelo = 80 cal/g

Troca de Calor

Qcedido + Qrecebido = 0

Princípio da conservação da energia: em um sistema isolado, a soma das quantidades de calor trocadas é zero.

Exemplo 1: Metal aquecido em água

Problema: Uma peça de ferro de 200 g é aquecida a 250 °C e é colocada em seguida em um recipiente com um litro (1 kg) de água a 25 °C. Depois de algum tempo, qual a temperatura de equilíbrio do sistema? (cferro = 0,11 cal/g°C, cágua = 1 cal/g°C)

Calor cedido pelo ferro:
Qferro = mferro · cferro · (Teq - Tif)
Calor recebido pela água:
Qágua = mágua · cágua · (Teq - Tia)
Equação de troca de calor:
Qcedido + Qrecebido = 0

Onde:
200 · 0,11 · (Teq - 250) + 1000 · 1 · (Teq - 25) = 0
22 · (Teq - 250) + 1000 · (Teq - 25) = 0
22Teq - 5500 + 1000Teq - 25000 = 0
1022 . Teq - 30500 = 0
1022 . Teq = 30500
Teq = 30500 / 1022
Teq ≈ 29,84 °C
Resposta: A temperatura de equilíbrio é aproximadamente 29,8 °C.

Legenda:
Qferro - calor cedido pelo ferro
Qágua - calor recebido pela água
mferro - massa de ferro
cferro - calor específico do Ferro
mágua - massa de água
cágua - calor específico da água
Teq - temperatura de equlíbrio
Tif - temperatura inicial do Ferro
Tia - temperatura inicial da água

Exemplo 2: Gelo em água (corrigido)

Problema: Uma pedra de gelo de 250 g, com temperatura inicial de -10 °C, é colocada em recipiente isolado com 1500 g de água a 25 °C. Calcule a temperatura de equilíbrio do sistema.
(cgelo = 0,5 cal/g°C, Lfusão = 80 cal/g, cágua = 1 cal/g°C)


1. Aquecimento do gelo de -10°C até 0°C:
 Q₁ = mgelo·cgelo·(0 -Tig) = 250·0,5·(0 - (-10)) = 250·0,5·10 = 1250 cal
2. Fusão total do gelo a 0°C:
 Q₂ = mgelo·Lfusão = 250·80 = 20000 cal
3. Aquecimento da água proveniente do gelo derretido (0°C até Teq):
 Q₃ = mgelo·cágua·(Teq - 0) = 250·1·Teq = 250·Teq
4. Resfriamento total da água (25°C até Teq):
 Q₄ = mágua·cágua·(Teq - 25) = 1500·1·(Teq - 25) = 1500·Teq - 37500
5. Equação de troca de calor:
 Q₁ + Q₂ + Q₃ + Q₄ = 0
1250 + 20000 + 250·Teq + 1500·(Teq - 25) = 0
21250 + 250Teq + 1500Teq - 37500 = 0
1750Teq - 16250 = 0
Teq = 16250 / 1750
Teq = 9,2857... °C
Resposta: A temperatura de equilíbrio é aproximadamente 9,29 °C. (Todo o gelo derrete e a mistura atinge equilíbrio térmico a 9,29°C)

Legenda:
Q1 - calor recebido pelo gelo para chegar a zero grau
Q2 - calor necessário para derreter o gelo
Q3 - calor recebido pelo gelo derretido
Q4 - calor total cedido pela água
mg - massa de gelo
cg - calor específico do gelo
mágua - massa de água
cágua - calor específico da água
Teq - temperatura de equlíbrio
Tig - temperatura inicial do gelo
Tia - temperatura inicial da água
Lfusão - calor latente de fusão do gelo

Exemplo 3: Capacidade térmica de um calorímetro

Problema: Calcule a capacidade térmica de um calorímetro (Cc) no qual contém 300 ml (300g) de água a 20 °C. Neste calorímetro é colocada uma barra de metal de massa 300 g, calor específico 0,2 cal/g°C, a 300 °C. A temperatura de equilíbrio do sistema é 50 °C. Despreze perdas.

Dados:
mágua=300g,
cágua=1 cal/g°C,
Tia=20°C,
mmetal=300g,
cmetal=0,2 cal/g°C,
Tim=300°C,
Teq=50°C,
Ccal = ?

Calor recebido pela água:
 Qágua = 300·1·(50-20) = 9000 cal
Calor recebido pelo calorímetro:
 Qcal = Ccal·(50-20) = 30·Ccal
Calor cedido pelo metal:
 Qmetal = 300·0,2·(50-300) = 60·(-250) = -15000 cal
Equação de troca de calor:
 Qágua + Qcalorímetro + Qmetal= 0
9000 + 30·Ccal - 15000 = 0
30·Cc = 6000
Ccal = 200 cal/°C
Resposta: A capacidade térmica do calorímetro é 200 cal/°C.

Legenda:
Qágua - calor recebido pela água
Qcal - calor recebido pelo calorímetro
Qmetal - calor cedido pelo metal
mmetal - massa de metal
cmetal - calor específico do metal
mágua - massa de água
cágua - calor específico da água
Teq - temperatura de equlíbrio
Tim - temperatura inicial do metal
Tia - temperatura inicial da água
Ccal - capacidade térmica do calorímetro
