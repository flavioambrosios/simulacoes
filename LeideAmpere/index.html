Lei de Ampere

B = (μ₀ · i) / (2πd)

Onde:

B → Campo magnético (T) ⟂ fio

μ₀ → 4π×10⁻⁷ T·m/A

i → Corrente elétrica (A)

d → Distância do fio (m)

Como interagir:

  • Arraste para rotacionar a câmera
  • Rolagem do mouse para zoom
  • Toque e arraste com um dedo para rotacionar
  • Pinça com dois dedos para zoom
