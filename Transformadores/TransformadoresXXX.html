Princípio do Transformador

Vs / Vp = Ns / Np

Onde:

Vp → Tensão primária

Vs → Tensão secundária

Np → Número de espiras primário

Ns → Número de espiras secundário

Como interagir:

  • Arraste para rotacionar a câmera
  • Rolagem do mouse para zoom
  • Toque e arraste com um dedo para rotacionar
  • Pinça com dois dedos para zoom
