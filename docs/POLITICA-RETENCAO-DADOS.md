# Política de Retenção e Governança de Dados

## Escopo
Esta política define regras para coleta mínima, armazenamento, retenção e descarte de dados do projeto Portal das Simulações.

## Princípios
- Minimização de dados: coletar apenas o necessário.
- Finalidade educacional e de pesquisa aplicada.
- Segurança e controle de acesso.
- Transparência e prestação de contas.

## Tipos de dados

### 1. Dados agregados pedagógicos
- Taxa de adesão.
- Taxa de conclusão.
- Desempenho médio em itens objetivos.
- Indicadores médios de respostas discursivas.

Retenção sugerida: até 5 anos para análise longitudinal.

### 2. Dados de trabalho transitórios
- Rascunhos de planilhas.
- Exportações intermediárias.

Retenção sugerida: até 6 meses.

### 3. Dados com potencial identificável (quando existirem)
- Qualquer campo que possa identificar estudante direta ou indiretamente.

Retenção sugerida: prazo mínimo necessário, com anonimização o quanto antes.

## Regras de armazenamento
- Armazenar dados em conta institucional sempre que possível.
- Manter planilhas com controle de acesso restrito.
- Evitar compartilhamento de links públicos de edição.
- Criar cópias de segurança apenas para dados essenciais.

## Controle de acesso
- Acesso preferencialmente limitado ao professor responsável e, quando necessário, à equipe autorizada.
- Revisão de permissões a cada semestre.

## Procedimento de anonimização
1. Remover nome, e-mail e qualquer identificador direto.
2. Agrupar por turma/série/período.
3. Revisar combinações de campos que possam reidentificar.
4. Publicar apenas estatísticas agregadas.

## Procedimento de descarte
- Exclusão de arquivos transitórios ao fim do prazo.
- Limpeza de lixeira e versões antigas, quando possível.
- Registro simples do descarte (data, tipo de dado, responsável).

## Auditoria mínima (checklist semestral)
- Há campos pessoais desnecessários sendo coletados?
- Existem arquivos antigos além do prazo de retenção?
- As permissões de acesso estão atualizadas?
- As publicações mantêm padrão agregado?

## Base normativa de referência
- Lei Geral de Proteção de Dados Pessoais (Lei 13.709/2018).
- Resolução CNS 510/2016 (pesquisa em Ciências Humanas e Sociais).
- Normativas institucionais da rede de ensino aplicável.
