# Checklist Rápido de Publicação

Use esta folha antes de compartilhar qualquer dado do projeto em slide, artigo, relatório, reunião ou conversa acadêmica.

## Conferência em 30 segundos

- [ ] Removi nomes de estudantes.
- [ ] Removi e-mails, telefones e matrículas.
- [ ] Os resultados estão em nível de turma, série ou período.
- [ ] Não há desempenho individual exposto.
- [ ] O nome da escola foi suprimido quando necessário.
- [ ] Revisei trechos discursivos para retirar pistas de identificação.
- [ ] O material está coerente com a política de dados do projeto.

## Regra prática

Se houver qualquer dúvida sobre reidentificação, não publicar até agregar mais os dados ou revisar novamente.
